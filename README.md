# pwy.io

This repository contains source code for [my website](https://pwy.io).

Powered by [Hugo](https://gohugo.io) and [Nix](https://nixos.org/).

## Building

```shell
$ nix build
```

## Running locally

```shell
$ nix develop
$ cd src
$ HUGO_NUMWORKERMULTIPLIER=1 hugo serve
```
