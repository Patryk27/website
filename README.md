# pwychowaniec.com

This repository contains source code for [my website](https://pwychowaniec.com).

Powered by [Hugo](https://gohugo.io) and [Nix](https://nixos.org/).

## Building

```shell
$ nix build
```

## Running locally

```shell
$ nix develop
$ cd src
$ hugo serve
```
